#### Before doing anything, make sure you've updated your glasses and the Meta AI app to the latest version, and that your glasses are in Developer mode. And of course... Make sure you're using a Meta Rayban Display glass!

Please access these resources to guide you with setting things up:
- https://wearables.developer.meta.com/docs/develop/webapps/setup/
- https://github.com/facebookincubator/meta-wearables-webapp

#### Steps to use this app on your glasses
1. Open "qr-test-on-device.png" and scan the QR code using your phone. This will automatically add the app to the Meta AI app.
2. Code to open the app is 289. You shouldn't be prompted to enter the code again when you have successfully done it once.
3. Enjoy!
